## Description

API to fetch [NEWS](https://gnews.io/dashboard) using caching.

## STEPS

1. Install redis
2. Create and add .env file as per the .env.example
3. run yarn
4. run yarn run start:dev

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
